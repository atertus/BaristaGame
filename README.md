# Barista Game

## Features

### 1. **One Drink at a Time Displayed**
   - The application displays one drink at a time for the user to interact with.

### 2. **User Input for Four Categories**
   - The user can select one option for each of the four categories: temperature, milk, syrup, and blended.
   - The selected option is displayed in the corresponding input box above the options.

### 3. **Option Replacement**
   - If an option is already in the input box, selecting a new option replaces the old one.

### 4. **Check Answer**
   - Clicking the "Check Answer" button submits the user's answer to be checked for correctness.
   - The application visually indicates which components of the drink were entered correctly and incorrectly by changing the background color of the input box.

### 5. **New Drink**
   - Clicking the "New Drink" button displays a new drink and clears the input boxes for the user to start fresh.

## How to Use

### 1. **Select Options**
   - Choose an option for each of the four categories: temperature, milk, syrup, and blended.
   - The selected option will appear in the input box above the options.

### 2. **Check Answer**
   - Once all options are selected, click the "Check Answer" button to see if your selections match the correct recipe.
   - The input boxes will change color to indicate correctness:
     - Green for correct answers
     - Red for incorrect answers

### 3. **New Drink**
   - Click the "New Drink" button to get a new drink and clear the input boxes.

## Styling

- The application uses Flexbox for layout and is styled to be visually appealing and user-friendly.
- The input boxes and buttons are styled to provide clear visual feedback to the user.

## Technologies Used

- React for building the user interface
- CSS for styling the application
- Flexbox for layout management
